This folder contains the table, tex, and run files for Tables 6-8 in the second draft (revision of the paper). These tables contain the experiments for the two-sided optimal omega vs optimal kappa comparison.


For example, to reproduce the table_6_Suite.tex, the user just needs to run table_6_Suite.m. Likewise, to reproduce table_7_Rand.tex, the user just needs to run table_7_Rand.m etc.
